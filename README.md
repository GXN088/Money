# Money
Расширь функциональность программы, которая позволит производить манипуляции с валютами.

В существующем абстрактном классе Money создай приватное поле amount типа double.
Создай публичный геттер для поля amount(public double getAmount()), чтобы к этому полю можно было получить доступ извне класса Money.
В отдельных файлах создай классы Hryvnia, Ruble и USD, которые будут являться потомками класса Money.
В классах Hryvnia, Ruble и USD реализуй метод getCurrencyName() который будет возвращать название соответствующей валюты (строку) в виде аббревиатуры (USD, UAH, RUB).
В классах Hryvnia, Ruble и USD реализуй публичный (public) конструктор, который принимает один параметр и вызывает конструктор базового класса (super) с этим параметром.
Заполни список allMoney объектами всех возможных в рамках условия задачи и функциональности программы валют.




Псевдокод:

```
// Абстрактный класс Money
Абстрактный класс Money:
    Приватное поле: amount (вещественное число)
    
    Конструктор Money(amount):
        Установить this.amount = amount
    
    Метод getAmount() -> вещественное число:
        Вернуть this.amount
    
    Абстрактный метод getCurrencyName() -> строка

// Класс Ruble наследует Money
Класс Ruble наследует Money:
    Конструктор Ruble(amount):
        Вызвать родительский конструктор с amount
    
    Переопределить метод getCurrencyName() -> строка:
        Вернуть "RUB"

// Класс USD наследует Money
Класс USD наследует Money:
    Конструктор USD(amount):
        Вызвать родительский конструктор с amount
    
    Переопределить метод getCurrencyName() -> строка:
        Вернуть "USD"

// Класс Hryvnia наследует Money
Класс Hryvnia наследует Money:
    Конструктор Hryvnia(amount):
        Вызвать родительский конструктор с amount
    
    Переопределить метод getCurrencyName() -> строка:
        Вернуть "UAH"

// Класс Person
Класс Person:
    Публичное поле: name (строка)
    Приватное поле: allMoney (список объектов Money)
    
    Конструктор Person(name):
        Установить this.name = name
        Создать новый пустой список allMoney
        Добавить в allMoney новый Hryvnia(2000)
        Добавить в allMoney новый USD(700)
        Добавить в allMoney новый Ruble(7000)
    
    Метод getAllMoney() -> список объектов Money:
        Вернуть allMoney

// Главный класс Solution
Класс Solution:
    Статический метод main(args):
        Создать объект ivan типа Person с именем "Иван"
        
        Для каждого объекта money в списке ivan.getAllMoney():
            Вывести на экран: 
                ivan.name + " имеет заначку в размере " + 
                money.getAmount() + " " + 
                money.getCurrencyName()
```

Объяснение структуры:

1. Абстрактный класс Money - базовый класс для валют
   · Содержит сумму денег
   · Определяет абстрактный метод для получения названия валюты
2. Конкретные классы валют (Ruble, USD, Hryvnia) - наследуют Money
   · Каждый возвращает свое название валюты
3. Класс Person - хранит список денег в разных валютах
   · При создании автоматически добавляет три вида валют
4. Главный класс Solution - демонстрирует работу
   · Создает объект Person
   · Выводит информацию о всех его деньгах в разных валютах
